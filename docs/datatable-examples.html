DataTable

ヘッダー付きの表を列名ベースで扱うためのクラス

Sample Data

共通サンプルデータ

idstatusamountowner
1OK120A
2NG80B
3OK210A
4WAIT95C

Example 1

状態が OK の行だけ抽出する

Dim tbl As New DataTable
Dim okRows As DataTable

tbl.read_range Sheet1.Range("A1:D5"), hasHeader:=True
Set okRows = tbl.filter_by_equals("status", "OK")
okRows.to_range Sheet1.Range("G1"), includeHeader:=True

レポート対象の行だけを残し、ヘッダーごと別の場所へ出力できます。

Example 2

派生列を追加する

Dim tbl As New DataTable

tbl.read_range Sheet1.Range("A1:D5"), hasHeader:=True
tbl.add_column "priority", Array("high", "low", "high", "mid")
tbl.to_range Sheet1.Range("G1"), includeHeader:=True

後続の判定や表示用に、元データにはない派生列を付け足せます。

Example 3

条件に合う行だけ値を更新する

Dim tbl As New DataTable

tbl.read_range Sheet1.Range("A1:D5"), hasHeader:=True
tbl.set_by_equals "status", "WAIT", "owner", "UNASSIGNED"
tbl.sort_by "amount", ascending:=False
tbl.to_range Sheet1.Range("G1"), includeHeader:=True

業務上のルールを列名ベースで書けるので、セル位置依存のコードを減らせます。
