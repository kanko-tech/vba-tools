DataTable

ヘッダー付きの表を列名ベースで扱うためのクラス

Class Outline

クラス概要

責務 ヘッダー付き表の読込、列名解決、抽出、更新、並び替え、出力
主な入力 ヘッダー付き Range、Matrix、列名配列
主な出力 DataTable、列配列、Vector、Range

Properties

主要プロパティ

名前 型 意味
is_loaded Boolean 表データが読み込まれているかを返します。
row_count Long データ行数を返します。
col_count Long 列数を返します。
column_names Variant 列名配列を返します。
matrix Matrix 内部データを Matrix として返します。

Method Groups

主要メソッド群

カテゴリ 主なメソッド 戻り値 / 効果
読込 read_range read_matrix ヘッダー情報つきの表をロードします。
列参照 col col_vector 指定列を配列または Vector として返します。
抽出 filter_by_mask filter_by_equals filter_by_in filter_by_contains filter_by_all_equals filter_by_any_equals select_columns 条件に合う DataTable を返します。
更新 add_column rename_column set_by_mask set_by_equals set_column 表の構造や値を更新します。
整列 sort_by 指定列を基準に並び替えます。
出力 to_range ヘッダー有無を選んでシートへ出力します。
状態管理 clear 内部状態を空に戻します。

Usage Notes

利用上の注意

  • read_range でヘッダーを読む場合、1 行目に重複しない列名がある前提です。
  • 列名ベースの API を使う前に、列名のゆれや余分な空白を整えておくと安全です。
  • 列の値を前処理したい場合は col_vector で Vector に渡すと責務が分かれます。
