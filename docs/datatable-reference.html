DataTable Reference

DataTable のプロパティとメソッドを、列名付き表データの操作単位で整理した技術詳細ページです。

Vector Matrix DataTable
Guide Reference

Loading

読込系

read_range

Sub read_range(ByVal rng As Range, Optional ByVal hasHeader As Boolean = True)

シート上の表を読み込みます。既定では 1 行目をヘッダー行として扱います。

read_matrix

Sub read_matrix(ByVal src As Matrix, ByVal columnNames As Variant)

Matrix と列名配列から DataTable を構築します。

Properties

参照系

is_loaded

Property Get is_loaded() As Boolean

データが読み込み済みかを返します。

row_count

Property Get row_count() As Long

本文の行数を返します。ヘッダー行は含みません。

col_count

Property Get col_count() As Long

列数を返します。

column_names

Property Get column_names() As Variant

列名の配列を返します。UI 表示や外部連携の確認に便利です。

matrix

Property Get matrix() As Matrix

内部表現を Matrix として返します。低レベルの処理へ戻したいときに使います。

col

Function col(ByVal columnName As String) As Variant

指定列を一次元配列として返します。

col_vector

Function col_vector(ByVal columnName As String) As Vector

指定列を Vector として返します。条件判定や集計と組み合わせやすくなります。

Filtering

抽出系

filter_by_mask

Function filter_by_mask(ByVal mask As Variant) As DataTable

ブール配列に対応する行だけを残した新しい DataTable を返します。

filter_by_equals

Function filter_by_equals(ByVal columnName As String, ByVal matchValue As Variant) As DataTable

指定列が一致する行だけを抽出します。

filter_by_in

Function filter_by_in(ByVal columnName As String, ByVal matchValues As Variant) As DataTable

候補配列のいずれかに一致する行を抽出します。

filter_by_contains

Function filter_by_contains(ByVal columnName As String, ByVal searchText As String, Optional ByVal caseSensitive As Boolean = False) As DataTable

部分一致で行を抽出します。テキスト検索向けです。

filter_by_all_equals

Function filter_by_all_equals(ByVal columnNames As Variant, ByVal matchValues As Variant) As DataTable

複数列の AND 条件で抽出します。

filter_by_any_equals

Function filter_by_any_equals(ByVal columnNames As Variant, ByVal matchValues As Variant) As DataTable

複数列の OR 条件で抽出します。

select_columns

Function select_columns(ByVal columnNames As Variant) As DataTable

必要な列だけを残した新しい DataTable を返します。

Updates

更新・整形系

add_column

Sub add_column(ByVal columnName As String, Optional ByVal values As Variant)

新しい列を追加します。値を省略した場合は空欄列として追加されます。

rename_column

Sub rename_column(ByVal oldName As String, ByVal newName As String)

列名を変更します。既存列との重複はできません。

sort_by

Sub sort_by(ByVal columnName As String, Optional ByVal ascending As Boolean = True)

指定列で並べ替えます。昇順・降順を切り替えられます。

set_by_mask

Sub set_by_mask(ByVal mask As Variant, ByVal columnName As String, ByVal newValue As Variant)

ブール配列が True の行だけを更新します。

set_by_equals

Sub set_by_equals(ByVal conditionColumnName As String, ByVal matchValue As Variant, ByVal targetColumnName As String, ByVal newValue As Variant)

条件列が一致した行だけ、対象列を更新します。

set_column

Sub set_column(ByVal columnName As String, ByVal values As Variant)

指定列全体を新しい値配列で置き換えます。

Output

出力系

to_range

Sub to_range(ByVal topLeft As Range, Optional ByVal includeHeader As Boolean = True)

指定セルからシートへ出力します。必要に応じてヘッダー行の有無を切り替えられます。

State

状態管理系

clear

Sub clear()

内部状態を初期化します。同じ変数を再利用したいときのリセット用です。
