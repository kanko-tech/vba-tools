DataTable

列名付き表データモジュールの API 詳細

DataTable は、ヘッダー行つきの表を列名ベースで扱うクラスです。 このページでは、クラスの位置づけ、プロパティ、各メソッドの入力例・コード例・実行結果を確認できます。

CLASS DESIGN

クラスデザイン

DataTable の位置づけ
DataTable は、本文データを Matrix として持ちながら、 列名一覧と列名マップを併用して「列名で読む・絞る・更新する」ためのクラスです。 セル番地や列番号ではなく、"status" や "amount" といった意味のある名前で処理を書きたい場面に向いています。
責務 ヘッダー付き表の列名ベース操作
主な入力 Range / Matrix / 列名配列 / 条件値
主な出力 DataTable / Vector / Matrix / シート出力

プロパティ

表データが読み込まれているかを返します。

現在の列数を返します。

Methods

メソッド

読込メソッド

表データを DataTable として読み込む入口です。

Methodread_range Sub read_range(ByVal rng As Range, Optional ByVal hasHeader As Boolean = True)Range から読む

シート上の範囲をヘッダー付き表として読み込みます。

入力例
idstatusamount
1OK100
2NG50
3OK80
コード例
Dim tbl As New DataTable
tbl.read_range Sheet1.Range("A1:C4"), hasHeader:=True
実行結果
tbl.row_count = 3
tbl.column_names = Array("id", "status", "amount")
Methodread_matrix Sub read_matrix(ByVal src As Matrix, ByVal columnNames As Variant)Matrix と列名配列から読む

本文データと列名を別々に持っている場合に使います。

入力例
src = [[1,"OK",100],[2,"NG",50]]
columnNames = Array("id", "status", "amount")
コード例
Dim tbl As New DataTable
tbl.read_matrix src, columnNames
実行結果
tbl.col_count = 3

取得メソッド

列や内部表現を取り出して他処理へつなぐためのメソッドです。

Methodcol Function col(ByVal columnName As String) As Variant列を配列で返す

列名指定で本文列を一次元配列として返します。

入力例
columnName = "amount"
コード例
values = tbl.col("amount")
実行結果
values = Array(100, 50, 80)
Methodcol_vector Function col_vector(ByVal columnName As String) As Vector列を Vector で返す

指定列を Vector として返し、そのまま条件判定や集計へ使えます。

入力例
columnName = "amount"
コード例
Dim vec As Vector
Set vec = tbl.col_vector("amount")
実行結果
vec.sum() = 230

抽出・選択メソッド

列名を基準に必要な行や列だけを取り出します。

Methodfilter_by_mask Function filter_by_mask(ByVal mask As Variant) As DataTableマスクで抽出

ブール配列に対応する行だけを残します。

入力例
mask = Array(True, False, True)
コード例
Set picked = tbl.filter_by_mask(mask)
実行結果
picked.row_count = 2
picked.col("id") = Array(1, 3)
Methodfilter_by_equals Function filter_by_equals(ByVal columnName As String, ByVal matchValue As Variant) As DataTable一致条件で抽出

指定列が一致する行だけを返します。

入力例
columnName = "status"
matchValue = "OK"
コード例
Set okRows = tbl.filter_by_equals("status", "OK")
実行結果
okRows.row_count = 2
okRows.col("id") = Array(1, 3)
Methodfilter_by_in Function filter_by_in(ByVal columnName As String, ByVal matchValues As Variant) As DataTable候補群で抽出

候補配列のいずれかに一致する行だけを返します。

入力例
matchValues = Array("OK", "HOLD")
コード例
Set rows = tbl.filter_by_in("status", matchValues)
実行結果
rows.row_count = 2
Methodfilter_by_contains Function filter_by_contains(ByVal columnName As String, ByVal searchText As String, Optional ByVal caseSensitive As Boolean = False) As DataTable部分一致で抽出

文字列表現が検索語を含む行だけを返します。

入力例
columnName = "status"
searchText = "O"
コード例
Set rows = tbl.filter_by_contains("status", "O")
実行結果
rows.col("id") = Array(1, 3)
Methodfilter_by_all_equals Function filter_by_all_equals(ByVal columnNames As Variant, ByVal matchValues As Variant) As DataTable複数列 AND 抽出

複数列条件をすべて満たす行を返します。

入力例
columnNames = Array("status", "amount")
matchValues = Array("OK", 100)
コード例
Set rows = tbl.filter_by_all_equals(columnNames, matchValues)
実行結果
rows.row_count = 1
rows.col("id") = Array(1)
Methodfilter_by_any_equals Function filter_by_any_equals(ByVal columnNames As Variant, ByVal matchValues As Variant) As DataTable複数列 OR 抽出

複数列条件のいずれかを満たす行を返します。

入力例
columnNames = Array("status", "amount")
matchValues = Array("NG", 80)
コード例
Set rows = tbl.filter_by_any_equals(columnNames, matchValues)
実行結果
rows.col("id") = Array(2, 3)
Methodselect_columns Function select_columns(ByVal columnNames As Variant) As DataTable列選択

必要な列名だけを残した新しい DataTable を返します。

入力例
columnNames = Array("id", "amount")
コード例
Set thin = tbl.select_columns(columnNames)
実行結果
thin.column_names = Array("id", "amount")

更新・整形メソッド

列追加、列名変更、並べ替え、値更新を行います。

Methodadd_column Sub add_column(ByVal columnName As String, Optional ByVal values As Variant)列追加

末尾に新しい列を追加します。

入力例
columnName = "flag"
values = Array("A", "B", "A")
コード例
tbl.add_column "flag", values
実行結果
tbl.column_names に "flag" が追加される
Methodrename_column Sub rename_column(ByVal oldName As String, ByVal newName As String)列名変更

既存列の名前を変更します。

入力例
oldName = "amount"
newName = "score"
コード例
tbl.rename_column "amount", "score"
実行結果
tbl.column_names = Array("id", "status", "score")
Methodsort_by Sub sort_by(ByVal columnName As String, Optional ByVal ascending As Boolean = True)並べ替え

指定列で本文行を並べ替えます。

入力例
columnName = "amount"
コード例
tbl.sort_by "amount", ascending:=False
実行結果
id の並び = Array(1, 3, 2)
Methodset_by_mask Sub set_by_mask(ByVal mask As Variant, ByVal columnName As String, ByVal newValue As Variant)条件行だけ更新

マスクが True の行だけ、対象列を更新します。

入力例
mask = Array(True, False, True)
columnName = "status"
newValue = "DONE"
コード例
tbl.set_by_mask mask, "status", "DONE"
実行結果
tbl.col("status") = Array("DONE", "NG", "DONE")
Methodset_by_equals Sub set_by_equals(ByVal conditionColumnName As String, ByVal matchValue As Variant, ByVal targetColumnName As String, ByVal newValue As Variant)一致条件で更新

条件列が一致した行だけ、対象列を書き換えます。

入力例
conditionColumnName = "status"
matchValue = "NG"
targetColumnName = "amount"
newValue = 0
コード例
tbl.set_by_equals "status", "NG", "amount", 0
実行結果
tbl.col("amount") = Array(100, 0, 80)
Methodset_column Sub set_column(ByVal columnName As String, ByVal values As Variant)列全体を置換

指定列の値を丸ごと差し替えます。

入力例
values = Array(110, 55, 88)
コード例
tbl.set_column "amount", values
実行結果
tbl.col("amount") = Array(110, 55, 88)

出力メソッド

加工した表をシートへ書き戻します。

Methodto_range Sub to_range(ByVal topLeft As Range, Optional ByVal includeHeader As Boolean = True)シート出力

現在の表をヘッダー付き、または本文のみでシートへ出力します。

入力例
topLeft = Sheet1.Range("E2")
includeHeader = True
コード例
tbl.to_range Sheet1.Range("E2"), includeHeader:=True
実行結果
EFG
idstatusamount
1OK100
2NG50
3OK80

状態管理メソッド

インスタンスを初期状態へ戻します。

Methodclear Sub clear()初期化

本文、列名、マップ、読込状態をまとめて消します。

入力例
tbl.is_loaded = True
tbl.row_count = 3
コード例
tbl.clear
実行結果
tbl.is_loaded = False
