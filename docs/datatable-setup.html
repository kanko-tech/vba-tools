DataTable

ヘッダー付きの表を列名ベースで扱うためのクラス

Import

取り込むファイル

DataTable.cls表形式ロジックの本体 Matrix.cls内部で行列を扱うために必要 Vector.cls列を Vector 化する API を使う場合に必要

Steps

導入手順

1

DataTable と依存クラスを VBE に取り込みます。

2

1 行目にヘッダーが入った矩形範囲を用意し、列名の重複を避けます。

3

read_range で読み込み、条件に合う行を filter_by_equals などで抽出します。

4

to_range で結果を書き戻し、ヘッダーごと確認します。

First Run

最初の実行コード

Sub DataTable_FirstRun()
    Dim tbl As New DataTable
    Dim result As DataTable

    tbl.read_range Sheet1.Range("A1:D6"), hasHeader:=True
    Set result = tbl.filter_by_equals("status", "OK")

    result.to_range Sheet1.Range("G1"), includeHeader:=True
End Sub

Checks

確認ポイント

  • ヘッダー行に空欄や重複名がないこと
  • 条件列の値の表記ゆれを事前にそろえておくこと
  • 列前処理が必要なら col_vector で Vector に渡すこと
