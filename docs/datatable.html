DataTable

ヘッダー付きの表を列名ベースで扱うためのクラス

Purpose

DataTable が向いている処理

入力 ヘッダー付きの Range または Matrix
得意な操作 列名解決、条件抽出、列追加、値更新、並び替え
主な出力 絞り込まれた DataTable、列配列、Range

「status 列が OK の行だけ残す」「amount 列を更新する」「必要な列だけ別表にする」といった、 列名を基準にした実務処理を明快に書きたいときに向いています。

When to Use

利用シーンの目安

列位置ではなく列名で書きたい

"status" や "amount" のような名前で処理を書くことで、列順の変更に強くなります。

抽出と更新を続けて行いたい

filter_by_equals、set_by_mask、sort_by などを同じ表オブジェクトの文脈で扱えます。

Vector / Matrix と役割分担したい

列単位の前処理は Vector、矩形の構造調整は Matrix、列名付きの業務ロジックは DataTable が担当します。

Quick Flow

最小フロー

Sub Sample_DataTable_Flow()
    Dim tbl As New DataTable
    Dim okRows As DataTable

    tbl.read_range Sheet1.Range("A1:D8"), hasHeader:=True
    Set okRows = tbl.filter_by_equals("status", "OK")

    okRows.to_range Sheet1.Range("A10"), includeHeader:=True
End Sub
