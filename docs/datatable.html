DataTable

DataTable は、ヘッダー行を持つ表データを列名ベースで扱うためのクラスです。 条件抽出、列の追加・更新、並べ替え、シートへの書き戻しを、セル座標ではなく列名中心で書けます。

Guide Reference

Overview

DataTable でできること

ヘッダー付き表の読込

read_range で、シート上の表を列名つきデータとして読み込めます。

列名ベースの抽出と更新

filter_by_equals や set_by_equals で、意味の分かるコードを書きやすくなります。

シートへの再出力

to_range でヘッダー込みの結果をそのまま書き戻せます。

Audience

向いている場面

列名つきの実務データを扱うとき 売上表、一覧表、CSV 由来のデータなど、列名を前提に処理を書くケースに向いています。
セル番地のハードコードを減らしたいとき "status" や "amount" を基準に処理できるため、保守時の見通しが良くなります。
Vector と Matrix を組み合わせたいとき 列単位の判定は Vector、内部の表構造は Matrix と連携して扱えます。

Setup

導入方法

1. インポート

DataTable.cls に加えて、依存する Vector.cls と Matrix.cls も取り込みます。

2. 読み込み

ヘッダー行を含む範囲を read_range へ渡し、列名と本文を読み込みます。

3. 加工

抽出、更新、並べ替え、列追加を列名ベースで組み合わせます。

4. 出力

to_range で、ヘッダー付きの結果をシートへ戻します。

Quick Start

最小例

以下は A1:D8 の表をヘッダー行付きで読み込み、 status = "OK" の行だけを抽出して A10 から書き出す例です。 

Sub Sample_DataTable_QuickStart()
    Dim tbl As New DataTable
    Dim okRows As DataTable

    tbl.read_range Sheet1.Range("A1:D8"), hasHeader:=True
    Set okRows = tbl.filter_by_equals("status", "OK")

    okRows.to_range Sheet1.Range("A10"), includeHeader:=True
End Sub

Benefits

使うメリット

読みやすい条件処理

filter_by_equals("status", "OK") のように、処理対象がそのままコードに現れます。

保守しやすい更新処理

列の並びが変わっても、列名が同じであればコードを大きく崩さずに済みます。
