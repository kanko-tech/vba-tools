Module Guide

DataTable

列名付きの表データを、列名ベースで読みやすく操作するためのクラスです。 実務データの抽出、更新、並べ替え、列追加を簡潔に書きたい場面に向いています。

Overview Vector Matrix DataTable

Use Cases

向いている場面

条件抽出

filter_by_equals や filter_by_in で列名ベースの抽出を簡潔に書けます。

条件付き更新

set_by_equals を使うと、条件に合う行だけ別列を更新できます。

実務向け整形

列追加、列名変更、並べ替えを組み合わせて、出力しやすいテーブルへ整えられます。

Quick Start

最小例

Sub Sample_DataTable_QuickStart()
    Dim tbl As New DataTable
    Dim okRows As DataTable

    tbl.read_range Sheet1.Range("A1:D8"), hasHeader:=True
    Set okRows = tbl.filter_by_equals("status", "OK")

    okRows.to_range Sheet1.Range("G1"), includeHeader:=True
End Sub

Methods

メソッドの見取り図

読込と参照

  • read_range: ヘッダー付き表を読む
  • read_matrix: `Matrix` に列名を付ける
  • column_names: 列名一覧
  • col / col_vector: 列取り出し

抽出

  • filter_by_mask: ブール配列で抽出
  • filter_by_equals: 一致抽出
  • filter_by_in: 候補一致抽出
  • filter_by_contains: 部分一致抽出

整形と更新

  • add_column: 列追加
  • rename_column: 列名変更
  • sort_by: 並べ替え
  • set_by_mask / set_by_equals: 条件更新

出力

  • to_range: ヘッダー付きで出力
  • clear: 状態リセット

Recipes

よくある使い方

`status="OK"` の行だけ抽出

Dim tbl As New DataTable
Dim okRows As DataTable

tbl.read_range Sheet1.Range("A1:D20"), hasHeader:=True
Set okRows = tbl.filter_by_equals("status", "OK")

`Vector` で条件を作って戻す

Dim tbl As New DataTable
Dim amountVec As Vector
Dim mask As Variant
Dim highRows As DataTable

tbl.read_range Sheet1.Range("A1:D20"), hasHeader:=True
Set amountVec = tbl.col_vector("amount")
mask = amountVec.gt(1000)

Set highRows = tbl.filter_by_mask(mask)
