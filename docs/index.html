VBA_tools

Excel VBA で使う再利用しやすいクラスモジュールをまとめたリポジトリです。 現在は Vector、Matrix、DataTable を中心に、 配列と表データの読み込み、加工、出力を扱える構成になっています。

Overview

このリポジトリでできること

配列処理を部品化

単一列の値をまとめて扱う Vector、二次元配列を表として扱う Matrix、列名つきの実務データを操作する DataTable を用意しています。

シート操作を整理

Range から読み込み、配列上で加工し、必要な形でシートへ戻す流れを クラス単位で整理できます。

Modules

主要モジュールの役割

Vector

  • 1 列または 1 行の Range を読み込む
  • 比較マスクを作る
  • 欠損補完や安全な型変換を行う
  • sum、mean、unique で集計する

Matrix

  • 矩形 Range を二次元配列として読む
  • 行フィルタ、行抽出、列選択を行う
  • 列追加や行追加で表を組み替える
  • 加工後の表をシートへ一括出力する

DataTable

  • ヘッダーつきの表を読む
  • 列名ベースでフィルタや更新を行う
  • 並べ替えや列追加を読みやすく書ける
  • Matrix と連携して業務データを扱う

Guide

使い分けの目安

Vector を選ぶ場面

単一列の値をまとめて整形したいときや、比較結果のマスクを作りたいとき。

Matrix を選ぶ場面

列名は不要で、表全体を二次元配列として扱いたいとき。

DataTable を選ぶ場面

列名つきの実務データに対して、条件抽出や列更新を読みやすく書きたいとき。

Quick Start

導入の流れ

  1. 必要なクラスモジュールを GitHub から取得します。
  2. Excel VBA プロジェクトへ Vector.cls、Matrix.cls、DataTable.cls をインポートします。
  3. Range から配列や表を読み込み、クラスメソッドで加工します。
  4. 加工結果をシートへ戻し、業務マクロの中で再利用します。
