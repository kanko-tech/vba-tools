Matrix

矩形データの読み込み、行抽出、列選択、転置のための 2 次元クラス

共通サンプルデータ

Example 1

マスクで行抽出する

Dim mat As New Matrix
Dim mask(1 To 4) As Boolean

mat.read_range Sheet1.Range("A2:C5")
mask(1) = True
mask(2) = False
mask(3) = True
mask(4) = False

mat.filter_rows(mask).to_range Sheet1.Range("F2")

別モジュールで作ったマスクを受けて、対象行だけを別領域へ展開できます。

Example 2

必要な列だけに絞る

Dim mat As New Matrix

mat.read_range Sheet1.Range("A2:C5")
mat.select_columns(Array(1, 3)).to_range Sheet1.Range("J2")

ID と Amount だけを抜き出して、報告用の表を別で作るような使い方に向きます。

Example 3

転置して比較しやすくする

Dim mat As New Matrix

mat.read_range Sheet1.Range("A2:C5")
mat.transpose.to_range Sheet1.Range("N2")

行方向に並ぶ比較表を作りたいときや、別モジュールへ渡す前に形を整えたいときに便利です。
