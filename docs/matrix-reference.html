Guide Reference

Loading

読込系

read_range

Sub read_range(ByVal rng As Range)

矩形の Range を読み込み、2 次元配列として保持します。

read_array

Sub read_array(ByVal arr As Variant)

2 次元配列をそのまま読み込みます。既存配列ロジックから移すときの入口です。

read_empty

Sub read_empty(ByVal colCount As Long)

列数だけを指定して、行数 0 の空の Matrix を作成します。

Properties

参照系

is_loaded

Property Get is_loaded() As Boolean

値が読み込まれているかを返します。

row_count

Property Get row_count() As Long

現在の行数を返します。

col_count

Property Get col_count() As Long

現在の列数を返します。

data

Property Get data() As Variant

内部の 2 次元配列を返します。

item

Function item(ByVal rowIndex As Long, ByVal colIndex As Long) As Variant

指定位置の値を返します。

row_values

Function row_values(ByVal rowIndex As Long) As Variant

指定行を一次元配列で返します。

col_values

Function col_values(ByVal colIndex As Long) As Variant

指定列を一次元配列で返します。

Selection

抽出・選択系

filter_rows

Function filter_rows(ByVal mask As Variant) As Matrix

ブール配列に対応する行だけを残した新しい Matrix を返します。

slice_rows

Function slice_rows(ByVal rowIndexes As Variant) As Matrix

指定行番号だけを順番どおりに取り出します。

select_columns

Function select_columns(ByVal columnIndexes As Variant) As Matrix

指定列だけを残した新しい Matrix を返します。

Mutation

更新・追加系

set_column_values

Sub set_column_values(ByVal colIndex As Long, ByVal values As Variant)

指定列の値を一括で置き換えます。

append_row

Sub append_row(ByVal values As Variant)

末尾へ 1 行追加します。

append_column

Sub append_column(ByVal values As Variant)

末尾へ 1 列追加します。フラグ列や計算列の追加に使えます。

Output

整形・出力系

transpose

Function transpose() As Matrix

転置した新しい Matrix を返します。

to_range

Sub to_range(ByVal topLeft As Range)

現在の表を指定セルからシートへ出力します。行数 0 の Matrix は出力できません。

State

状態管理系

clear

Sub clear()

内部状態を初期化します。再利用前のリセット用です。
