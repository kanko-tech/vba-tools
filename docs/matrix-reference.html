Matrix

二次元表データモジュールの API 詳細

Matrix は、列名を持たない表データを二次元配列として扱う基盤クラスです。 このページでは、クラスの位置づけ、プロパティ、各メソッドの入力例・コード例・実行結果を確認できます。

CLASS DESIGN

クラスデザイン

Matrix の位置づけ
Matrix は、列名を持たない表データを二次元配列として保持し、 行抽出、列選択、追加、転置、出力といった低レベルの表操作を受け持つクラスです。 DataTable が列名ベースの処理を行う一方で、Matrix は配列構造そのものを扱う土台として使います。
責務 表データの保持、行列操作、配列ベースの出力
主な入力 Range / 二次元配列 / 行番号配列 / 列番号配列
主な出力 Matrix / 配列 / シート出力

Methods

メソッド

読込メソッド

シートや二次元配列から Matrix を作る入口です。

Methodread_range Sub read_range(ByVal rng As Range)矩形 Range を読む

シート上の矩形範囲を二次元配列として読み込みます。

入力例
AB
12
34
コード例
Dim mat As New Matrix
mat.read_range Sheet1.Range("A1:B2")
実行結果
mat.row_count = 2
mat.col_count = 2
Methodread_array Sub read_array(ByVal arr As Variant)二次元配列を読む

既存の二次元配列をそのまま Matrix に読み込みます。

入力例
arr(1,1)=1  arr(1,2)=2
arr(2,1)=3  arr(2,2)=4
コード例
Dim mat As New Matrix
mat.read_array arr
実行結果
mat.data(2, 2) = 4
Methodread_empty Sub read_empty(ByVal colCount As Long)空の Matrix を作る

列数だけを持つ、行数 0 の Matrix を初期化します。

入力例
colCount = 3
コード例
Dim mat As New Matrix
mat.read_empty 3
実行結果
mat.row_count = 0
mat.col_count = 3

抽出・選択メソッド

必要な行や列だけを切り出した新しい Matrix を返します。

Methodfilter_rows Function filter_rows(ByVal mask As Variant) As Matrixマスクで行抽出

ブール配列に対応する行だけを残します。

入力例
AB
1020
3040
5060
mask = Array(True, False, True)
コード例
Set filtered = mat.filter_rows(mask)
実行結果
AB
1020
5060
Methodslice_rows Function slice_rows(ByVal rowIndexes As Variant) As Matrix指定順で行切り出し

行番号配列の順番どおりに行を取り出します。

入力例
rowIndexes = Array(3, 1)
コード例
Set sliced = mat.slice_rows(rowIndexes)
実行結果
1行目 = 元の3行目
2行目 = 元の1行目
Methodselect_columns Function select_columns(ByVal columnIndexes As Variant) As Matrix列選択

必要な列番号だけを残します。

入力例
columnIndexes = Array(2, 1)
コード例
Set selected = mat.select_columns(columnIndexes)
実行結果
列順が B, A に入れ替わった Matrix が返る

更新・追加メソッド

既存の Matrix を更新したり、行や列を追加したりします。

Methodset_column_values Sub set_column_values(ByVal colIndex As Long, ByVal values As Variant)列全体を置換

指定列を新しい値配列で置き換えます。

入力例
values = Array("A", "B", "C")
コード例
mat.set_column_values 2, values
実行結果
2列目が "A", "B", "C" に更新される
Methodappend_row Sub append_row(ByVal values As Variant)行追加

末尾に 1 行を追加します。

入力例
values = Array(70, 80)
コード例
mat.append_row values
実行結果
最終行に 70, 80 が追加される
Methodappend_column Sub append_column(ByVal values As Variant)列追加

末尾に 1 列を追加します。

入力例
values = Array("X", "Y", "Z")
コード例
mat.append_column values
実行結果
右端に X, Y, Z の列が追加される

出力メソッド

mat.data = [[1,2],[3,4]]

12
34
