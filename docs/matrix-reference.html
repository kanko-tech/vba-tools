Matrix

矩形データの読み込み、行抽出、列選択、転置のための 2 次元クラス

Class Outline

クラス概要

責務 2 次元データの読込、抽出、列選択、形状変更、出力
主な入力 Range、2 次元配列、列数付き空行列
主な出力 Matrix、Range

Properties

主要プロパティ

名前 型 意味

col_count Long 列数を返します。

Usage Notes

利用上の注意

  • 行抽出に使うマスクの長さは行数と一致させます。
  • 列選択や列更新は 1 始まりの列番号で指定する前提です。
  • 列名ベースの処理に進む場合は、整形後に DataTable へ受け渡すと責務が明確になります。
