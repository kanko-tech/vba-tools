Tool Box VBA_tools Excel VBA Matrix 導入方法

Matrix

矩形データの読み込み、行抽出、列選択、転置のための 2 次元クラス

Import

取り込むファイル

Matrix.cls矩形データ専用のクラスです

Steps

導入手順

1

クラスモジュールをインポートし、名前が Matrix であることを確認します。

2

読み込む矩形範囲を用意し、空白行や列を含めるか事前に決めます。

3

read_range で読み込み、必要ならマスクや列番号配列を準備します。

4

to_range や row_count / col_count で結果を確認します。

First Run

最初の実行コード

Sub Matrix_FirstRun()
    Dim mat As New Matrix
    Dim mask(1 To 3) As Boolean

    mat.read_range Sheet1.Range("A2:C4")
    mask(1) = True
    mask(2) = False
    mask(3) = True

    mat.filter_rows(mask).to_range Sheet1.Range("F2")
End Sub

Checks

確認ポイント

  • 抽出に使うマスクの長さが行数と一致していること
  • 列番号は 1 始まりで指定すること
  • 列名ベースの処理へ進むなら、この段階で DataTable へ渡すかを決めること
