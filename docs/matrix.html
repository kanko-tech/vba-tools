Matrix

矩形データの読み込み、行抽出、列選択、転置のための 2 次元クラス

Purpose

Matrix が向いている処理

入力 矩形の Range、2 次元配列、空行列
得意な操作 行抽出、列選択、行列の追加、転置
主な出力 新しい Matrix、または Range への書き戻し

シート上の範囲をいったん読み込んでから、必要な行だけ残し、 列を絞って別の場所へ出力する。そうした 2 次元処理の中間状態を安全に扱いたいときに役立ちます。

When to Use

利用シーンの目安

表の一部だけを抜き出したい

ブールマスクによる行抽出や、列番号指定による列選択を段階的に進めたいときに向きます。

シートと処理ロジックを分けたい

シート上の範囲操作を最初と最後に閉じ込め、中間の変換は Matrix 内で完結できます。

DataTable の土台にしたい

最終的に列名付き表へ進めたい場合も、まずは Matrix で形を整えてから DataTable.read_matrix に渡せます。

Quick Flow

最小フロー

Sub Sample_Matrix_Flow()
    Dim mat As New Matrix
    Dim filtered As Matrix
    Dim picked As Matrix
    Dim mask(1 To 4) As Boolean

    mat.read_range Sheet1.Range("A2:C5")

    mask(1) = True
    mask(2) = False
    mask(3) = True
    mask(4) = False

    Set filtered = mat.filter_rows(mask)
    Set picked = filtered.select_columns(Array(1, 3))
    picked.to_range Sheet1.Range("F2")
End Sub
