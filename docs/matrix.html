Module Guide

Matrix

列名を持たない表データを 2 次元配列として保持し、抽出・選択・追加・出力まで扱う基本クラスです。 `DataTable` より一段低いレベルで表操作を組み立てたいときに向いています。

Overview Vector Matrix DataTable

Use Cases

向いている場面

表全体を配列で扱う

列名なしの矩形データを一括で読み込み、配列中心で加工したいときに向いています。

中間表を作る

行抽出や列選択で必要な部分だけを切り出し、次段の処理へ渡せます。

`DataTable` の土台にする

`Matrix` に列名を付けて `DataTable` に持ち上げる流れにも使えます。

Quick Start

最小例

Sub Sample_Matrix_QuickStart()
    Dim mat As New Matrix
    Dim filtered As Matrix
    Dim picked As Matrix
    Dim mask(1 To 4) As Boolean

    mat.read_range Sheet1.Range("A2:C5")

    mask(1) = True
    mask(2) = False
    mask(3) = True
    mask(4) = False

    Set filtered = mat.filter_rows(mask)
    Set picked = filtered.select_columns(Array(1, 3))

    picked.to_range Sheet1.Range("F2")
End Sub

Methods

メソッドの見取り図

読込と参照

  • read_range: 矩形 `Range` を読む
  • read_array: 二次元配列を読む
  • read_empty: 空表を作る
  • row_count / col_count: 形状確認

抽出と選択

  • filter_rows: 行マスクで抽出
  • slice_rows: 指定行だけ抜き出す
  • select_columns: 必要列だけ選ぶ

更新と追加

  • set_column_values: 列全体を更新
  • append_row: 行追加
  • append_column: 列追加

変形と出力

  • transpose: 転置した新しい表を返す
  • to_range: シートへ一括出力
  • clear: 状態リセット

よくある使い方

行抽出して必要列だけ残す

Dim mat As New Matrix
Dim filtered As Matrix
Dim picked As Matrix
Dim mask(1 To 4) As Boolean

mat.read_range Sheet1.Range("A2:D5")
mask(1) = True
mask(2) = False
mask(3) = True
mask(4) = False

Set filtered = mat.filter_rows(mask)
Set picked = filtered.select_columns(Array(1, 4))

列を追加して `DataTable` へ渡す

Dim mat As New Matrix
Dim tbl As New DataTable

mat.read_range Sheet1.Range("A2:C6")
mat.append_column Array("OK", "NG", "OK", "OK", "NG")

tbl.read_matrix mat, Array("date", "item", "amount", "status")
