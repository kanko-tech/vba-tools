Module Guide

Matrix

Matrix は 2 次元配列を扱うための基本クラスです。 列名を持たない表データを読み込み、行抽出、列選択、追加、転置、シート出力を組み合わせて扱えます。

Guide Reference

Overview

Matrix でできること

表の読み込み

read_range や read_array で、列名なしの表データをそのまま 2 次元配列として保持できます。

行列の切り出し

filter_rows slice_rows select_columns で必要な部分だけを抜き出せます。

更新と出力

append_row append_column to_range を使って加工結果を戻せます。

When to Use

向いている場面

列名より表構造を重視したいとき シンプルに「行と列の塊」として扱いたい処理に向いています。
DataTable より一段低いレベルで組み立てたいとき 行抽出や列追加を配列ベースで組みたい場合の土台として使いやすいです。
最終的に列名をあと付けしたいとき 途中までは Matrix で加工し、最後に DataTable.read_matrix へ渡す流れにも向いています。

Setup

導入方法

1. インポート

src/ExcelVBA/Matrix/Matrix.cls を VBA プロジェクトへ取り込みます。

2. 読み込み

read_range でシート上の表を読み込みます。ヘッダーは値の一部として扱われます。

3. 加工

行抽出、列選択、列追加などを組み合わせて必要な形へ整えます。

4. 出力

to_range で結果を書き戻すか、DataTable の入力に渡します。

Quick Start

最小例

以下は A2:C5 の表を読み込み、ブール配列で行を抽出したあと、 1 列目と 3 列目だけを残して F2 へ書き出す例です。 

Sub Sample_Matrix_QuickStart()
    Dim mat As New Matrix
    Dim filtered As Matrix
    Dim picked As Matrix
    Dim mask(1 To 4) As Boolean

    mat.read_range Sheet1.Range("A2:C5")

    mask(1) = True
    mask(2) = False
    mask(3) = True
    mask(4) = False

    Set filtered = mat.filter_rows(mask)
    Set picked = filtered.select_columns(Array(1, 3))

    picked.to_range Sheet1.Range("F2")
End Sub

Workflow

よくある使い方

中間処理の土台にする

ヘッダーを意識せずにまず表を整えたいときは、Matrix のまま加工を進めたほうが単純です。

DataTable へ引き渡す

列名が必要になった段階で DataTable.read_matrix に渡せば、責務を分けて扱えます。
