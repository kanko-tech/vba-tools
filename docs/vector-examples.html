Example 1

閾値を超える行のマスクを作る

Dim vec As New Vector
Dim mask As Variant

vec.read_col_range Sheet1.Range("B2:B6")
vec.fill_empty 0
vec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True
mask = vec.gt(100)

出力は Array(True, False, False, True, False) のような Boolean 配列です。Matrix や DataTable の抽出条件としてそのまま使えます。

Example 2

空欄を補完して合計と平均を出す

Dim vec As New Vector

vec.read_col_range Sheet1.Range("B2:B6")
vec.fill_empty 0
vec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True

Debug.Print vec.sum
Debug.Print vec.mean

売上や件数のような列を、レポート前に軽く整形して集計したいときの最小パターンです。

Example 3

整形した列を別領域へ出力する

Dim vec As New Vector

vec.read_col_range Sheet1.Range("B2:B6")
vec.fill_empty "(missing)"
vec.cast_to_string_safe
vec.to_range_vertical Sheet1.Range("D2")

入力列をそのまま壊さず、別領域に整形済みの結果を並べたいときに使います。
