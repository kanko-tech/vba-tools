Guide Reference

Loading

読込系

read_col_range

Sub read_col_range(ByVal rng As Range)

1 列の Range を読み込みます。縦方向の値をそのまま内部配列へ格納します。

用途 シート上の列データを Vector 化したいとき
注意 複数列の範囲は想定外です

read_row_range

Sub read_row_range(ByVal rng As Range)

1 行の Range を読み込みます。横方向の値を内部配列として保持します。

用途 見出し行や横並びの設定値を扱うとき
注意 複数行の範囲は対象外です

read_array

Sub read_array(ByVal arr As Variant)

一次元配列をそのまま読み込みます。既存の配列処理から Vector に移したいときの入口です。

用途 VBA 内で作った配列をクラスメソッドで扱いたいとき
注意 二次元配列は受け付けません

Properties

参照系

is_loaded

Property Get is_loaded() As Boolean

値が読み込まれているかを返します。事前チェックやデバッグ向けです。

data

Property Get data() As Variant

内部の一次元配列を返します。他モジュールや自前ロジックへ値を渡すときに使います。

count

Property Get count() As Long

要素数を返します。ループや空チェックの基準になります。

item

Function item(ByVal index As Long) As Variant

指定位置の値を返します。個別確認や条件分岐の補助に使えます。

type_names

Property Get type_names() As Variant

各要素の TypeName を返します。データ品質の確認に向いています。

Conditions

条件マスク系

eq

Function eq(ByVal matchValue As Variant) As Variant

一致判定のブール配列を返します。最も基本的な絞り込み条件です。

ne

Function ne(ByVal matchValue As Variant) As Variant

不一致判定のブール配列を返します。

gt / ge

Function gt(ByVal threshold As Variant) As Variant
Function ge(ByVal threshold As Variant) As Variant

しきい値より大きい、または以上の条件を作ります。数値列の判定によく使います。

lt / le

Function lt(ByVal threshold As Variant) As Variant
Function le(ByVal threshold As Variant) As Variant

しきい値より小さい、または以下の条件を作ります。

is_empty

Function is_empty(Optional ByVal treatBlankStringAsEmpty As Boolean = True) As Variant

空欄判定のブール配列を返します。空文字を空欄扱いするかも指定できます。

Transform

変換・更新系

cast_to_double_safe

Sub cast_to_double_safe(Optional emptyAsZero As Boolean = False, Optional invalidAsZero As Boolean = False, Optional treatDateAsInvalid As Boolean = True)

値を安全に Double へ変換します。空欄や変換失敗時の扱いも指定できます。

cast_to_string_safe

Sub cast_to_string_safe(Optional emptyAsBlank As Boolean = True, Optional errorAsBlank As Boolean = True)

値を文字列へ揃えます。表示用や比較前の正規化に向いています。

cast_to_date_safe

Sub cast_to_date_safe(Optional ByVal invalidAsEmpty As Boolean = True)

日付として解釈できる値を Date に変換します。

fill_empty

Sub fill_empty(ByVal fillValue As Variant, Optional ByVal treatBlankStringAsEmpty As Boolean = True)

空欄や空文字を指定値で埋めます。数値化や集計の前処理として使いやすいです。

map

Sub map(ByVal functionName As String)

各要素へ同じ関数を適用して値を更新します。ルールベースの変換を共通化したいときに向いています。

Aggregate

集計系

unique

Function unique() As Variant

重複を除いた値の配列を返します。カテゴリ一覧の作成などに使えます。

sum

Function sum() As Double

数値要素の合計を返します。事前に cast_to_double_safe を通すと安定します。

mean

Function mean() As Double

数値要素の平均を返します。集計結果の確認用に使いやすいメソッドです。

Output

出力系

to_range_vertical

Sub to_range_vertical(ByVal topLeft As Range)

値を縦方向へ書き出します。列として結果を確認したいときに向いています。

to_range_horizontal

Sub to_range_horizontal(ByVal topLeft As Range)

値を横方向へ書き出します。見出し行や簡易確認用の出力に向いています。

State

状態管理系

clear

Sub clear()

内部状態を初期化します。同じインスタンスを再利用したいときのリセット用です。
