Vector

一次元データ操作モジュールの API 詳細

Vector は、一次元データを読んで、条件判定・変換・集計・出力までを行うクラスです。 このページでは、クラスの位置づけ、プロパティ、メソッドの使い方を、入力例と結果つきで確認できます。

Guide Reference

CLASS DESIGN

クラスデザイン

Vector の位置づけ
Vector は 1 列または 1 行ぶんの値を、そのまま一次元配列として保持する軽量クラスです。 行列全体を扱う Matrix や、列名付き表を扱う DataTable よりも粒度が小さく、 「特定列だけを条件判定したい」「数値列だけを集計したい」といった局面で活きます。
責務 一次元データの読込・条件判定・変換・集計
主な入力 Range / 一次元配列
主な出力 配列 / Double / シート出力

Properties

プロパティ

Property is_loaded Property Get is_loaded() As Boolean 読込済みかを返す

値が読み込まれているかを返します。事前チェック用です。

Property data Property Get data() As Variant 内部配列を返す

内部の一次元配列を複製して返します。

Property count Property Get count() As Long 要素数を返す

現在保持している要素数を返します。

Property item Property Get item(ByVal index As Long) As Variant 指定位置の値を返す

インデックス指定で個別の値を返します。

Methods

メソッド

読込メソッド

シートや配列から一次元データを読み込む入口です。

Method read_col_range Sub read_col_range(ByVal rng As Range) 1 列の Range を読む

縦方向の 1 列範囲を読み込みます。

入力例
A
10
20
30
コード例 
Dim vec As New Vector
vec.read_col_range Sheet1.Range("A1:A3")
実行結果 
vec.count = 3
vec.data = Array(10, 20, 30)
Method read_row_range Sub read_row_range(ByVal rng As Range) 1 行の Range を読む

横方向の 1 行範囲を読み込みます。

入力例
ABC
456
コード例 
Dim vec As New Vector
vec.read_row_range Sheet1.Range("A1:C1")
実行結果 
vec.count = 3
vec.data = Array(4, 5, 6)
Method read_array Sub read_array(ByVal arr As Variant) 一次元配列を読む

既存の一次元配列をそのまま読み込みます。

入力例
arr = Array("A", "B", "C")
コード例
Dim vec As New Vector
vec.read_array arr
実行結果
vec.data = Array("A", "B", "C")

確認メソッド

中身の型や値の状態を把握するためのメソッドです。

Method type_names Function type_names() As Variant 型名の配列を返す

各要素の型を文字列で返します。

入力例
vec.data = Array(10, "A", Empty)
コード例
Dim types As Variant
types = vec.type_names()
実行結果
types = Array("Integer", "String", "Empty")

条件判定メソッド

抽出に使うブール配列を作るメソッド群です。

Methodeq Function eq(ByVal matchValue As Variant) As Variant一致マスク

値が一致する要素だけ True にします。

入力例

コード例
mask = vec.eq("OK")
実行結果

Methodne Function ne(ByVal matchValue As Variant) As Variant不一致マスク

変換・更新メソッド

値の型や内容を整えるためのメソッド群です。

Methodcast_to_double_safe Sub cast_to_double_safe(...)Double へ変換

数値化できる値を安全に Double へ変換します。

入力例
vec.data = Array("10", "", "30")
コード例
vec.cast_to_double_safe emptyAsZero:=True
実行結果
vec.data = Array(10#, 0#, 30#)
Methodcast_to_string_safe Sub cast_to_string_safe(...)文字列へ変換

全要素を比較しやすい文字列へそろえます。

入力例
vec.data = Array(100, Empty, 300)
コード例
vec.cast_to_string_safe
実行結果
vec.data = Array("100", "", "300")
Methodcast_to_date_safe Sub cast_to_date_safe(...)Date へ変換

日付として読める値だけを Date にします。

入力例
vec.data = Array("2026/04/01", "not date", "2026/04/03")
コード例
vec.cast_to_date_safe
実行結果
vec.data = Array(#2026/04/01#, Empty, #2026/04/03#)
Methodfill_empty Sub fill_empty(ByVal fillValue As Variant, ...)空欄を埋める

空欄や空文字を指定値で埋めます。

入力例
vec.data = Array("", Empty, "A")
コード例
vec.fill_empty "(未設定)"
実行結果
vec.data = Array("(未設定)", "(未設定)", "A")
Methodmap Sub map(ByVal functionName As String)関数を一括適用

各要素に同じ変換関数を適用します。

入力例
vec.data = Array(" A ", "B ", " C")
コード例
' TrimValue(value, index) を別途定義
vec.map "TrimValue"
実行結果
vec.data = Array("A", "B", "C")
