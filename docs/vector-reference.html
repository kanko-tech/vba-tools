Vector

1 次元データの読み込み、条件マスク、型変換、集計のためのクラス

Class Outline

クラス概要

責務 1 次元データの読み込み、条件判定、型変換、集計、書き戻し
主な入力 Range、1 次元配列
主な出力 Variant 配列、Double、Range

Properties

主要プロパティ

名前 型 意味
is_loaded Boolean データが読み込まれているかを返します。
data Variant 内部の 1 次元配列を返します。
count Long 要素数を返します。
item(index) Variant 指定位置の要素を返します。

Usage Notes

利用上の注意

  • 読込前に集計や判定を呼ぶと成立しないため、先に read_* 系を実行します。
  • 数値集計前は cast_to_double_safe や fill_empty で前提をそろえるのが安全です。
  • 戻り値のマスクは Variant 配列なので、そのまま Matrix や DataTable の抽出に流用できます。
