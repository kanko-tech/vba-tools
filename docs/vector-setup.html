Vector

1 次元データの読み込み、条件マスク、型変換、集計のためのクラス

Import

取り込むファイル

Vector.clsこのモジュールだけで単独利用できます

Steps

導入手順

1

VBE でクラスモジュールをインポートし、名前が Vector であることを確認します。

2

シート上に 1 列または 1 行のサンプルデータを用意します。

3

read_col_range か read_row_range を呼び、読み込みます。

4

count や to_range_vertical で正しく読み込めたか確認します。

First Run

最初の実行コード

Sub Vector_FirstRun()
    Dim vec As New Vector

    vec.read_col_range Sheet1.Range("B2:B5")
    vec.fill_empty 0

    Debug.Print vec.count
    vec.to_range_vertical Sheet1.Range("D2")
End Sub

Checks

確認ポイント

  • 対象 Range が 1 列または 1 行になっていること
  • 数値集計前に必要なら fill_empty や cast_to_double_safe を通すこと
  • 別モジュールへ渡す場合は戻り値のマスクの長さをそろえること
