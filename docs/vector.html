Module Guide

Vector

単一列または単一行のデータを、比較・整形・集計しやすい形で扱うためのクラスです。 条件マスク生成や数値列の前処理をまとめたいときに向いています。

Overview Vector Matrix Table

Use Cases

向いている場面

列を一括整形

空欄補完、数値変換、文字列変換などを 1 列単位でまとめて処理できます。

条件マスク作成

eq や gt で作ったマスクを、`Matrix` や `Table` の抽出に渡せます。

集計の前処理

sum や mean の前に安全な型変換を挟めるので、実務列の整形に向いています。

Quick Start

最小例

Sub Sample_Vector_QuickStart()
    Dim vec As New Vector
    Dim mask As Variant

    vec.read_col_range Sheet1.Range("B2:B6")
    vec.fill_empty 0
    vec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True

    mask = vec.gt(100)

    Debug.Print vec.sum
    Debug.Print vec.mean
End Sub

Methods

メソッドの見取り図

読込

  • read_col_range: 1 列の `Range` を読む
  • read_row_range: 1 行の `Range` を読む
  • read_array: 一次元配列を読む

参照

  • data: 配列コピーを返す
  • count: 要素数を返す
  • item: 指定要素を返す
  • type_names: 型名一覧を返す

条件マスク

  • eq / ne: 一致・不一致判定
  • gt / ge: 下限比較
  • lt / le: 上限比較
  • is_empty: 欠損判定

整形と集計

  • cast_to_double_safe: 数値化
  • cast_to_string_safe: 文字列化
  • fill_empty: 欠損補完
  • sum / mean / unique: 集計

Recipes

よくある使い方

空欄を 0 にして合計する

Dim vec As New Vector

vec.read_col_range Sheet1.Range("D2:D8")
vec.fill_empty 0
vec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True

Debug.Print vec.sum

`Table` と組み合わせる

Dim tbl As New Table
Dim scoreVec As Vector

tbl.read_range Sheet1.Range("A1:D10"), hasHeader:=True
Set scoreVec = tbl.col_vector("score")

scoreVec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True
Debug.Print scoreVec.mean
