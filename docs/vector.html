Module Guide

Vector

Vector は 1 列または 1 行の値を、まとめて読み込み、条件マスク作成や型変換、集計に使うためのクラスです。 Excel の列データを VBA で扱う処理を、セル単位の繰り返しより短く明確に書けます。

Overview

Vector でできること

列データの読み込み

read_col_range と read_row_range で、シート上の 1 列または 1 行をそのまま取り込めます。

条件マスクの作成

eq gt is_empty などで、後続処理に渡しやすいブール配列を作れます。

型変換と集計

cast_to_double_safe や sum mean で、実務データの整形と集計をまとめて行えます。

Audience

向いている場面

1 列の値をまとめて扱いたいとき セルを 1 つずつ読むのではなく、列全体を値の配列として扱いたいケースに向いています。
条件判定だけを独立させたいとき しきい値判定や空欄判定をマスクとして取り出し、Matrix や DataTable の抽出へ渡せます。
数値化や平均計算をまとめたいとき 文字列混じりの実務データを安全に変換し、そのまま集計までつなげられます。

Setup

導入方法

1. インポート

src/ExcelVBA/Vector/Vector.cls を VBA プロジェクトへ取り込みます。

2. 読み込み

read_col_range または read_row_range で対象範囲を読み込みます。

3. 加工

必要に応じて空欄補完、型変換、条件判定、集計を行います。

4. 連携

マスクを別モジュールへ渡すか、to_range_vertical などでシートへ戻します。

Quick Start

最小例

以下は B2:B6 の列を読み込み、空欄を 0 で補完したうえで数値化し、 合計と平均を出力する例です。 

Sub Sample_Vector_QuickStart()
    Dim vec As New Vector
    Dim mask As Variant

    vec.read_col_range Sheet1.Range("B2:B6")
    vec.fill_empty 0
    vec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True

    mask = vec.gt(100)

    Debug.Print vec.sum
    Debug.Print vec.mean
End Sub

Workflow

よくある使い方

マスク作成の部品として使う

eq や gt で作ったブール配列は、Matrix.filter_rows や DataTable.set_by_mask の入力に使えます。

集計前の整形に使う

空欄や文字列を含む列でも、fill_empty と cast_to_double_safe を先に通すことで、その後の計算が安定します。
