Vector

1 次元データの読み込み、条件マスク、型変換、集計のためのクラス

Purpose

Vector が向いている処理

入力 縦 1 列、横 1 行、または 1 次元配列
得意な操作 条件マスク、空値補完、型変換、集計
主な出力 Boolean 配列、集計値、縦横どちらかの Range

シート上の 1 列データを読み込み、空欄を埋め、数値へ変換し、閾値判定のマスクを作る。 その一連の処理を Excel のセル操作ではなくクラスのメソッドでまとめたいときに使います。

When to Use

利用シーンの目安

列単位の前処理をしたい

売上列やステータス列のように、ひとつの系列に対して同じ処理をまとめて適用したい場面に向きます。

マスクを他モジュールへ渡したい

eq や gt で作った Boolean 配列を、Matrix.filter_rows や DataTable.filter_by_mask に流し込めます。

集計前に型をそろえたい

cast_to_double_safe や fill_empty を先に通すことで、合計や平均の前提を明示できます。

Quick Flow

最小フロー

Sub Sample_Vector_Flow()
    Dim vec As New Vector
    Dim mask As Variant

    vec.read_col_range Sheet1.Range("B2:B8")
    vec.fill_empty 0
    vec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True

    mask = vec.gt(100)

    Debug.Print vec.sum
    Debug.Print vec.mean
End Sub
